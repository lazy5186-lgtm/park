# [Hook & Style Rules]
1. **도입부 공식**: `[키워드] + [경험/신뢰] + [알려줄게]` 패턴
   - ✅ "○○○, 직접 해보고 효과 봤던 것들만 솔직하게 정리해드릴게요"
   - ❌ "오늘은 ○○○에 대해 알아보겠습니다"
2. **금지 어미**: "~알아보겠습니다", "~이 가장 이상적입니다", "꼭 확인하세요" 등
3. **추상 표현 금지 → 수치 의무**: "저렴하다" → "1만원대"
4. **꿀팁/1인칭 경험 1회 이상 의무**

# [Naver Blog SEO Rules]
1. 키워드를 제목 1회, 서론 1회, 본문 3~4회, 결론 1회
2. 한 단락 3~4줄 이하
3. 핵심 문장 **볼드체** 강조
4. `[📸 이곳에 관련 이미지 1장 삽입]` 6~10회
5. 글 마지막에 해시태그 5~8개
